This project is a beginner-friendly cybersecurity tool that checks the strength of a password.
It analyzes the password based on common security rules and classifies it as **Weak**, **Medium**, or **Strong**.

Strong passwords are important to protect user accounts from attacks such as brute-force and password guessing.
